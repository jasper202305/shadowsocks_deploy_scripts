# shadowsocks_deploy_scripts
VPN Deploy Scripts

The current VPN deployment version is shadowsocks v1.24.0

### 部署脚本
- `wget -qO- https://github.com/jasper202305/shadowsocks_deploy_scripts/blob/main/install.sh | bash`
### 重置密码
- `wget -qO- https://github.com/jasper202305/shadowsocks_deploy_scripts/blob/main/reset_password.sh | bash`
### 查看密码
- `cat /opt/shadowsocks/config.json`
### 各版本客户端下载
- `https://github.com/shadowsocks`
